# ExpenseTracker-Python

python program to track personal expenses and visualize them with graphs.

This project uses Python 3, pandas for data handling and matplotlib for visualization


## Features

- Add new expenses with the following optional fields:
   Date 
   Category 
   Amount 
   Description
- Automatically saves expenses to a CSV file (`expenses.csv`)
- Visualize total expenses by category using a bar chart

## Requirements

- Python 3.x
- [pandas](https://pandas.pydata.org/) (for csv files)
- [matplotlib](https://matplotlib.org/) (for graphs)

## Install required packages
pip install pandas matplotlib
